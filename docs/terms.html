Description of Service

CodeSentinel is an open-source Slack application that analyzes software codebases for dead code, circular dependencies, coupling issues, and architectural drift. The App provides analysis results within Slack conversations and optionally via an MCP (Model Context Protocol) server. CodeSentinel is provided as-is and is intended for use in professional software development environments.
