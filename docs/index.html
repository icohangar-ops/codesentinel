Dead Code Detection

Finds unused functions, classes, modules, and exports. Each finding includes file path, line number, severity, and fix suggestions.

Circular Dependency Analysis

DFS-based cycle detection across module imports. Shows the exact import chain, impact assessment, and remediation strategy.

Coupling Metrics

Measures fan-out per module, identifies tightly coupled clusters, and provides refactoring suggestions like Facade or DIP.

Architectural Drift Detection

Checks for layer boundary violations: UI importing from data, business layer breaching into UI, and shared module contamination.

Full Health Scan

Runs all four analyses and computes a 0–100 health score with letter grade, category breakdown, and AI-generated executive summary.

MCP Server

Standalone MCP server with 6 analysis tools. Works with Claude Desktop, Cursor, Windsurf, or any MCP-compatible client.
